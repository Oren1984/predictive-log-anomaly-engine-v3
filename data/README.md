# Data

This directory contains datasets used by the anomaly detection pipeline.


## Structure:

data/
├── raw/        # Original log datasets
├── processed/  # Structured events used by the models

raw data is downloaded or generated by scripts such as:

scripts/10_download_data.py


## Processed data

Processed data is created by preprocessing stages including:

- event parsing
- template extraction
- sequence generation

Processed datasets are used for training and runtime evaluation.
