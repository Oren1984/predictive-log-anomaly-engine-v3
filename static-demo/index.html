Architecture

System Overview

Three-layer architecture — V1 ensemble baseline, V2 deep learning pipeline, and optional V3 semantic enrichment layer

Log Data Processing ML Model Alert System API Layer Observability
⚡
Triple-Layer Architecture

V1 (ensemble baseline) and V2 (deep learning) run in parallel; V3 semantic enrichment adds optional explanation and similarity scoring on top of confirmed alerts

🔒
Per-Stream Isolation

Separate rolling buffers per service/session — a noisy stream never pollutes another's baseline

📡
Observability-First

Prometheus metrics, health checks, and Grafana dashboards built in — not bolted on
